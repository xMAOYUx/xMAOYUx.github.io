
# Personal Site (Static, Minimal)

Minimalist HTML/CSS/JS personal website in English with light/dark theme, CV download, Blog, and Media pages. Ready for GitHub Pages.

## Quick Start (Local)
Open `index.html` directly or use a local server (e.g., VS Code Live Server).

## Deploy to GitHub Pages
1. Create a repo named `your-username.github.io` on GitHub.
2. Upload all files in this folder to the `main` branch.
3. In **Settings → Pages**, choose **Deploy from a branch**, select branch `main`, folder `/root`.
4. Wait 1–3 minutes and visit `https://your-username.github.io`.

## Customize
- **Texts/Links**: edit `index.html`, `blog.html`, `media.html`, and files in `posts/`.
- **CV**: replace `assets/cv.pdf` with your own CV file (same name).
- **Colors/Theme**: tweak CSS variables in `style.css` under `:root` and `[data-theme="dark"]`.
- **Avatar/Favicon**: replace `assets/avatar.svg` and `assets/favicon.svg`.
